# E-Commerce Data Analysis Project

Proyek ini bertujuan untuk menganalisis tren penjualan dan segmentasi pelanggan
menggunakan dataset Brazilian E-Commerce (Olist).

## Tools
- Python
- Pandas
- Matplotlib
- Streamlit

## Cara Menjalankan Dashboard
Dashboard dapat dijalankan secara lokal menggunakan Streamlit.  
Dashboard juga telah dideploy dan dapat diakses melalui tautan berikut:
